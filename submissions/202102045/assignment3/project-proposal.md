# Project Proposal: Lecture Navigator (강의 자료 이중 언어 분석기)

## 1. 프로젝트 개요 (Project Overview)

| 항목 | 설명 |
| :--- | :--- |
| **프로젝트명** | Lecture Navigator (강의 자료 이중 언어 분석기) |
| **주요 목표** | 영어 강의 자료(PPT 슬라이드)를 자동으로 번역, 요약, 핵심 키워드 추출 및 학습 자료로 가공하여 학습 효율 극대화 |
| **주요 기능** | 영-한 번역 및 요약, 영-한 병기 키워드 추출, 연습 문제 생성, 연관 기사 추천 |
| **ML 학습 목표** | 핵심 키워드 영-한 병기 추출 모델 직접 학습 및 평가 |
| **실사용성** | 모든 영어 전공 수업에 적용 가능하며, 최소 5회 이상 사용 보장 |

---

## 2. 해결하고자 하는 문제 

### 2.1. 본인이 겪고 있는 실제 문제

저는 국제학 등 **모든 전공 수업이 영어로 진행**되어 학습 자료(주로 PPT 슬라이드)를 이해하는 데 상당한 비효율성을 겪고 있습니다.

* **반복적인 수동 번역:** 슬라이드를 이해하기 위해 문장 단위로 번역기를 사용하는 과정이 반복되어 학습 흐름이 자주 끊깁니다.
* **이중 언어 키워드 관리의 어려움:** 개념 이해는 한국어로 해야 하지만, **시험은 영어**로 치러야 하므로 핵심 키워드를 **영-한 병기**로 정리하는 데 시간이 많이 소요됩니다.
* **최신 현황 파악 부담:** 국제학의 특성상 수업 자료 외에 **최근 뉴스와 현황**을 찾아보는 것이 필수이나, 슬라이드 주제별로 관련 기사를 수동으로 검색하는 것이 번거롭습니다.

### 2.2. 이 도구를 본인이 실제로 사용할 것인가?

**네, 이 도구는 저의 전공 필수 학습 도구가 될 것입니다.** 새로운 수업 자료가 나올 때마다 자동 분석을 요청하여 시간을 절약할 것이며, **최소 5회 이상** 반복적으로 사용할 것입니다.

## 3. 제안하는 솔루션

### 3.1. 주요 기능 및 기대 효과

| 기능 | 설명 | 기대 효과 |
| :--- | :--- | :--- |
| **이중 언어 번역 및 요약** | 영어 슬라이드 전체를 **영-한 병기** 형태로 번역하고, 전체 내용을 세부적으로 요약하여 제공합니다. | 수업 내용 이해 속도 획기적으로 향상 |
| **핵심 키워드 영-한 병기 추출** | 슬라이드에서 가장 중요한 키워드를 **영어 원문과 한국어 번역을 짝지어** 목록으로 제공합니다. | 시험 대비 노트 정리 및 개념 암기 효율화 |
| **맞춤형 연습 문제 생성** | 슬라이드 내용을 기반으로 **객관식/주관식 연습 문제**를 자동 생성합니다. | 자가 학습 및 시험 준비 과정 지원 |
| **최근 연관 기사 제안** | 슬라이드 주제와 밀접하게 관련된 **최근 영어 기사/보고서** 링크를 추천합니다. | 최신 정보 파악 및 리서치 시간 절약 |

### 3.2. 사용자 사용 흐름

1.  **자료 입력:** 사용자(본인)가 PPT 슬라이드 텍스트 또는 PDF 내용을 도구에 업로드/입력합니다.
2.  **자동 분석:** 도구가 입력된 텍스트를 분석하여 네 가지 핵심 결과물(번역/요약, 영-한 키워드, 문제, 연관 기사)을 동시에 생성합니다.
3.  **학습 활용:** 사용자는 생성된 결과물을 바탕으로 효율적인 학습 및 시험 대비를 진행합니다.

## 4. ML 활용 계획 (ML Application Plan)

본 프로젝트는 영어 텍스트를 다루는 복합적인 NLP 도구이지만, **Assignment 5의 목표인 ML 모델 직접 학습 경험**에 중점을 두고 계획을 수립합니다.

### 4.1. Assignment 5 ML 모델 학습 목표

#### **핵심 키워드 영-한 병기 추출 모델 학습 (Bilingual Keyword Extractor)**

* **ML 목표:** 수업 자료 텍스트가 주어졌을 때, 문맥에서 중요한 **핵심 키워드**를 자동으로 추출하고 이에 대한 정확한 한국어 번역(레이블)을 부여하는 지도 학습 모델을 학습합니다.
* **학습 경험:** 텍스트 전처리, 특징 추출(TF-IDF), 그리고 지도 학습 모델(예: Logistic Regression 또는 SVM)을 사용한 이진 분류/시퀀스 라벨링 경험을 목표로 합니다.

### 4.2. 활용할 데이터 수집 계획

| 항목 | 세부 계획 |
| :--- | :--- |
| **데이터 유형** | **영어 텍스트** (학술 개념), **한국어 번역 쌍** (정의된 키워드) |
| **수집 방법** | **국제학/사회과학 분야의 용어집(Glossary)**, 교재 및 논문의 **핵심 정의 문장**을 수집합니다. |
| **레이블링** | 수집한 문장 내에서 중요한 개념을 **영어 키워드**로 추출하고, 이에 대한 정확한 **한국어 번역**을 수동으로 레이블링하여 학습 데이터셋을 구축합니다. |

### 4.3. ML 모델 및 기술 스택

* **벡터화:** **TF-IDF** (Term Frequency-Inverse Document Frequency)
* **모델:** 나이브 베이즈(Naïve Bayes) 또는 선형 분류기(Linear Classifier)
* **최종 도구 구현 (Assignment 6):** 직접 학습한 모델을 기반으로 하되, 번역/요약/문제 생성과 같은 복잡한 작업의 품질 향상을 위해 **공개 LLM API** (예: Gemini API 등)를 통합하여 최종 기능을 구현할 것입니다.
