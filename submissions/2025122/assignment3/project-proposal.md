# 3차 과제 프로젝트 제안서

## 필요성
- arxivbot 에서 모든 논문을 다 요약하는게 불편하다. 비싸기도 하고, 요약을 했다지만 1000편의 논문의 요약본을 읽는 것도 굉장히 부담스럽다.
- 전체 논문이 아니라, 한번 내 관심사에 맞게 필터링된 논문들에 대해서만 요약을 받아보고 싶다.
- 전체 논문을 한번 필터링 할 수 있는 relevance classification 모델을 학습시키고자 한다.

## 데이터 수집
- 이를 위한 데이터 수집은, 기존의 arxiv 논문 text 를 입력으로 받아서, 나랑 관계 있는 (relevant = 1) 혹은 관계 없는 (relevance = 0) 을 분류하는 모델을 만들고자 한다.
- 데이터 수집은 csv 형태로 모을거고, 첫번째 컬럼에는 (전처리된 논문 text)가 들어갈 것이고, 두번째 컬럼에는 내가 직접 레이블링한 relevance (0 혹은 1) 을 갖도록 저장할 것이다.
